# LoL Teamfight Manager

ローカルで動作する軽量な Vue 風 UI（カスタム実装）で、リーグ・オブ・レジェンドの試合構成とチャンピオン使用状況を管理するアプリです。

## 機能

- 試合情報（チーム名、対戦相手、サイド、勝敗、使用チャンピオン、メモ、日付）の登録
- チャンピオンはリストから選択式で追加（最大 5 体）
- 登録済み試合のフィルタリング（チーム名・対戦相手・サイド・勝敗）
- 全試合を集計し、チャンピオンの使用回数を「合計 / 勝利 / 敗北」でソート表示
- ローカルストレージへ自動保存

## 使い方

1. リポジトリ直下で簡易サーバを立ち上げます。

   ```bash
   python -m http.server 4173
   ```

2. ブラウザで `http://localhost:4173/public/index.html` にアクセスしてください。
3. 「試合を登録」フォームから必要事項を入力し、チャンピオンを選択して登録します。
4. 「チャンピオン使用状況」パネルで使用回数を確認し、ソートボタンで並び替えられます。
5. 試合リストはフィルターを活用して検索できます。

> **補足**: 外部 CDN を利用せず、`src/lib/vue-lite.js` に実装した最小限の Vue 互換 API（`createApp`/`h`）で構築しています。
